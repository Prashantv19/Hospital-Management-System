# 🏥 Hospital Management System (Java)

## 📌 Overview
The **Hospital Management System** is a simple Java-based console application designed to manage basic hospital operations such as:

- Patient registration
- Doctor management
- Appointment scheduling

It demonstrates core **Object-Oriented Programming (OOP)** concepts like classes, objects, encapsulation, and data handling using collections.

---

## 🚀 Features
- ✅ Add new patients  
- ✅ Add new doctors  
- ✅ Schedule appointments  
- ✅ View patient records  
- ✅ View doctor details  
- ✅ View all appointments  

---

## 🛠️ Technologies Used
- Java (Core Java)  
- ArrayList (Java Collections Framework)  
- OOP Concepts  
- Console-based UI  

---

## 📂 Project Structure
```bash
HospitalManagement.java   # Main driver class
Patient.java              # Patient entity
Doctor.java               # Doctor entity
Appointment.java          # Appointment entity
```

---

## ⚙️ How It Works

1. User runs the program  

2. Menu is displayed:
```text
1. Add Patient
2. Add Doctor
3. Schedule Appointment
4. View Patients
5. View Doctors
6. View Appointments
0. Exit
```

3. User selects an option  

4. Data is stored in memory using `ArrayList`  

---

## ▶️ How to Run

### Step 1: Compile
```bash
javac *.java
```

### Step 2: Run
```bash
java HospitalManagement
```
